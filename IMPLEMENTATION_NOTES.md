# Teddy Bear Website Notes

## Goal
- Build a cute teddy-bear-themed website for girls, with soft pastel visuals.

## Implemented
- Created a responsive one-page website with:
  - Hero section and custom typography
  - Hero photo frame using user's cute girl image
  - Seasonal skin switcher (Candy, Noel, Valentine)
  - Teddy card collection with heart interactions
  - Love note generator button
  - Teddy mini game area with:
    - Catch-the-bear score game (15 seconds)
    - High score persistence via localStorage
    - Level progression with increasing speed
    - Confetti celebration on new high score
    - Daily performance stats tracking
    - Themed daily event mission board and reward claim
    - Weekly mission board with weekly reward claim
    - Daily lucky wheel reward mechanic
    - Honey Coin economy across mini-games
    - Daily streak bonus system with coin multiplier
    - Lucky number guessing game (1 to 5)
    - Memory flip-card mini-game
    - Memory combo mechanic with bonus rewards
    - Image tile puzzle mini-game using user-provided photo
    - Puzzle difficulty modes (3x3 and 4x4) with timer and best-time records
    - Sticker catching game using photo thumbnails
    - Image quadrant guessing game
    - Photo rotation alignment game
    - Dress-up teddy game with expanded accessories (dress, bag, necklace)
    - Random outfit generator
    - Dress-up state persistence via localStorage
    - Theme-exclusive event accessory by seasonal skin
    - Reward inventory list (persistent)
    - Achievement badges (bronze/silver/gold) based on catch high score
    - Teddy care module with mood meter (persistent)
    - Second pet care module (buddy mood, persistent)
    - Teddy shop with coin-based reward purchases
    - Chapter quest map with unlock and one-time claim rewards
    - Local leaderboard (catch best, memory best, coin best)
    - Weekly and monthly coin ranking metrics
    - Weekend mini boss challenge with high coin reward
    - Toggleable game sound effects
  - Optional ambient music toggle
- Added meaningful motion with staged reveal animation.
- Built mobile-friendly layout behavior.

## Files
- index.html
- styles.css
- script.js

## Design Direction
- Palette: peach, strawberry pink, mint, cream.
- Font pairing: Baloo 2 + Quicksand.
- Atmosphere: playful, soft, dreamy.
