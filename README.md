# Bearie Land

Bearie Land la mot website game mini phong cach gau bong de thuong, su dung HTML, CSS, JavaScript thuan.

## Noi dung chinh

- Giao dien pastel de thuong, responsive mobile va desktop.
- Hero section co hinh anh nguoi dung cung cap.
- Nhieu mini game tuong tac:
  - Bat Gau 15 Giay
  - Doan So May Man
  - Mix Do Cho Gau Bong
  - Event ngay va event tuan
  - Vong quay may man
  - Cham soc 2 pet (gau + tho)
  - Lat the gau bong
  - Puzzle hinh anh (3x3, 4x4, timer, best time)
  - Mini boss cuoi tuan
- He thong Honey Coin, inventory, chapter map, badge, leaderboard local.

## Cau truc file

- index.html
- styles.css
- script.js
- IMPLEMENTATION_NOTES.md
- README.md

## Chay du an

Du an la static site, mo truc tiep file index.html trong trinh duyet la co the choi.

## Ghi chu

- Du lieu game duoc luu trong localStorage trinh duyet.
- De reset tien trinh, xoa du lieu localStorage cua trang.
